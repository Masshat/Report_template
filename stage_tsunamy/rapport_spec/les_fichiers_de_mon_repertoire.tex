\chapter{Les fichiers de mon répertoire au laboratoire}

Vous devez faire à l'attention de votre encadrant, une annexe décrivant de manière détaillée des fichiers produits pendant votre stage et un mode d'emploi détaillé qui permet d'utiliser votre réalisation.
   \begin{itemize}
   \item
La documentation que vous avez utilisée : où est-elle ? Où est le fichier .bib qui regroupe les références ?
   \item
Votre algorithme : est-il bien décrit dans le rapport ? où sont les calculs intermédiaires ?
   \item
Les fichiers que vous avez créés : où sont-ils ? Comment sont-ils organisés ? Où est le Makefile ?
   \item
Les outils extérieurs que vous avez utilisés : la liste et le mode d'emploi pour les utiliser.
   \item
Les articles ou documentations annexes que vous avez rédigés, le cas échéant, pendant votre stage : sont-ils signalés dans votre rapport ?

   \item
Les figures que vous avez réalisées pendant votre stage : où sont-elles ?

   \item
Les divers fichiers de simulation que vous avez créés pendant votre stage : où sont-ils ? Où est le README permettant de les réutiliser ?
   \end{itemize}
